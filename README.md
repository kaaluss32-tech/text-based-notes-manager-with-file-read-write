# Notes Manager (Java)

A simple **text-based Notes Manager** in Java that allows you to add, view, and delete notes. Notes are stored in a text file (`notes.txt`).

## Features

- Add new notes
- View all saved notes
- Delete all notes
- Persistent storage using a text file
